# AlertaHidrica

Gráficos de pronóstico horario de precipitaciones y altura de isoterma-0 para los próximos 2 días, referidos al centro geométrico de las comunas de Chile.

Los gráficos se agruparon en carpetas, una por región de Chile, y adentro puede descargar el archivo png de la comuna de interés. Si no ve la lista de carpeta de las regiones, favor hacer click en "view code"

- Para ver resultados del pronóstico meteorológico y de caudales, y mediciones hidrometeorológicas en tiempo real: https://explora.alertahidrica.cl

- Para conocer acerca del proyecto de innovación social Alerta Hídrica: http://www.alertahidrica.cl/
